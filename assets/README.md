# assets

여기에 포트폴리오에서 링크할 첨부파일을 넣으세요.

권장 구성:
```
assets/
├── ppt/
│   ├── 01-주제발표.pdf
│   ├── 02-중간발표.pdf
│   └── 03-최종발표.pdf
└── 요구사항명세서.pdf
```

- 발표자료는 **PDF로 변환**해서 넣는 걸 권장합니다(브라우저에서 바로 열림, .pptx는 다운로드만 됨).
- 파일을 넣은 뒤 `js/main.js`의 `CONFIG.resources`에서 해당 항목 `href`에 경로를 지정하세요.
  예: `href: "assets/ppt/03-최종발표.pdf"`

웨어러블 로봇 디바이스의 입고 · 임대 · A/S · 반납 전 생애주기를 관리하고, MQTT로 착용자의 생체 · 응급 데이터를 실시간 수신하는 사내 운영 시스템입니다. 11라운드에 걸친 설계 개선과 OWASP 기반 보안 대응을 거쳤습니다.

About

프로젝트 개요

단일 운영 센터 내부에서 디바이스 자산과 임대 흐름, 그리고 착용자 안전 데이터를 한곳에서 관리합니다.

디바이스 생애주기 관리

입고 → 임대대기 → 임대중 → A/S → 반납/폐기까지 상태 전이 규칙(validateStatusTransition)으로 잘못된 전이를 차단합니다.

실시간 데이터 수신

DrValue MQTT 라이브러리로 착용자의 생체 리포트와 응급 신호를 자동 수신해 DB에 적재합니다.

운영 보안

JWT 인증, 역할 기반 접근통제(RBAC), 로그인 잠금, 시크릿 외부화 등 OWASP Top 10을 코드 레벨에서 대응했습니다.

Tech Stack

사용 기술

Spring Boot 4 + JPA 기반 백엔드, Thymeleaf 서버사이드 렌더링, 원격 MySQL.
